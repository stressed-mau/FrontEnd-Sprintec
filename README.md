# FrontEnd-Sprintec

##  Descripción del Proyecto

Este proyecto corresponde al desarrollo del frontend de un sistema web generador de portafolios de proyectos de software, desarrollado en el marco de la materia **Taller de Ingeniería de Software (TIS)**.

El sistema permitirá a los usuarios crear, gestionar y visualizar su portafolio profesional, incluyendo proyectos, habilidades y logros.


## Tecnologías Utilizadas

*  React (Vite)
*  React Router DOM (navegacion)
*  Axios (conexion con backend Laravel)
*  Tailwind CSS / Frameworks UI 


## Estructura del Proyecto

src/
├── assets/           # imágenes, íconos
├── components/       # componentes reutilizables
├── pages/            # páginas (Login, Dashboard, etc.)
├── layouts/          # estructura general (Navbar, Sidebar)
├── services/         # conexión con backend (API)
├── hooks/            # lógica reutilizable
├── context/          # estado global (auth, usuario)
├── routes/           # rutas del sistema
└── App.jsx

##  Instalación y Ejecución

1. Clonar repositorio:

git clone https://github.com/stressed-mau/FrontEnd-Sprintec.git

2. Instalar dependencias:

npm install

3. Ejecutar proyecto:

npm run dev
